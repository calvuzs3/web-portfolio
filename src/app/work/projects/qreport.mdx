## Overview

Quick Report is a professional Android application designed for field technicians who manage
inspection and maintenance reports on robotic cells and industrial plants.
The app focuses on **offline-first** workflows: all data lives on the device
and syncing to a remote server is entirely optional.

> 🚧 **Version 1.0.0 — coming soon on Google Play**, currently in internal testing.

## Key Features

- **Inspection reports** — create, edit and export structured reports for each plant or robotic cell.
- **Photo attachments** — capture photos directly from the camera or attach existing files.
- **Client & plant registry** — manage a local database of clients, sites and equipment.
- **Maintenance log** — track intervention history per asset.
- **Optional remote sync** — connect to your own Ktor/PostgreSQL server; disabled by default.
- **Fully offline** — no account required, no data sent anywhere without explicit user action.

## Privacy

Quick Report collects **no personal data**. All content stays on the user's device.
Remote sync, if activated, connects only to the server configured by the user —
the developer has no access to any user data.

→ [Full Privacy Policy](/work/qreport/privacy-policy)

## Technologies Used

- **Kotlin + Jetpack Compose** — modern Android UI toolkit.
- **Room / SQLite** — local database for offline-first storage.
- **EncryptedSharedPreferences** — secure storage for optional server credentials.
- **Ktor Client** — HTTP client for optional remote synchronisation.
- **Kotlin Coroutines + Flow** — reactive data layer.
- **CameraX** — camera integration for photo capture.
